## Downloading Datasets
Run the following in your terminal:

```bash
cd scripts
```

```bash
chmod +x download_dataset.sh
```

```bash
./download_dataset.sh
```

## Downloading Models
Run the following in your terminal:

```bash
cd scripts
```

```bash
chmod +x download_model.sh
```

```bash
./download_model.sh
```

## Download Requirements
Run the following in your terminal:

```bash
pip install -r requirements.txt
```
